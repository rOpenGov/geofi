# Municipality level population data from Sotkanet

This dataset contains population data at municipality level pulled from
THL (Sotkanet) from 2000 to 2022

## Usage

``` r
sotkadata_population
```

## Format

A data frame with 7107 rows and 3 variables:

- municipality_code:

  municipality_code

- primary.value:

  primary.value

- year:

  year
